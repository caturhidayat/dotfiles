# Install STOW

install GNU STOW first.
on macos :
`brew install stow`

- for linux you can find on the internet.

## Structure folder for STOW or Conventional naming

**dotfiles**
~/.config/nvim \n
~/.config/tmux \n
~/.zshrc \n

**GNU STOW**
nvim/.config/nvim \n
tmux/.config/tmux \n
zsh/.zshrc \n

## Use STOW

cd to dotfiles dir, and then :

`stow [dir name]`
