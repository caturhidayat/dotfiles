# Install STOW

install GNU STOW first.
on macos :
`brew install stow`

- for linux you can find on the internet.

## Structure folder for STOW or Conventional naming

## **dotfiles**

- ~/.config/nvim
- ~/.config/tmux
- ~/.zshrc

## **GNU STOW**

- nvim/.config/nvim
- tmux/.config/tmux
- zsh/.zshrc

## Use STOW

## STOW Package

cd to dotfiles dir, and then :

`stow [dir name]`

## UNSTOW Package

`stow -D [dir name]`
